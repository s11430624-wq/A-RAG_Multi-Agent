# Manual Task Packets

每個資料夾是一題可直接丟給 Hermes 的題目包。

使用方式：

- A/C：只給對應題目的 task_packet.md，不要給 RAG corpus。
- E：給對應題目的 task_packet.md，再加上同題的 manual_rag_corpus/Txx/。
- 每個 repetition 都要開新 session。
- hidden test 結果只能最後記錄，不可以回饋給 Hermes。
